# Презентация дипломного проекта

## Habitify - Трекер привычек

### Актуальность
Формирование полезных привычек требует систематического контроля. Существующие решения либо слишком сложные, либо платные. Цель проекта - создать доступный трекер с базовым бесплатным функционалом и премиум-возможностями.

### Целевая аудитория
- Люди, желающие выработать полезные привычки
- Студенты, следящие за учебными целями
- Профессионалы для рабочих рутин

### Реализованный функционал

**Базовый (Free):**
- До 5 привычек
- Отметка выполнения (факт/количество)
- Базовая статистика
- Реклама

**Premium:**
- Безлимит привычек
- Расширенная аналитика
- Без рекламы
- Экспорт данных

### Технические решения

**Backend:**
- MVC паттерн для разделения логики
- Кастомный роутер вместо фреймворка
- PDO для работы с БД
- Сессии для аутентификации

**Frontend:**
- Bootstrap 5 для адаптивности
- Axios для AJAX
- Chart.js для визуализации

**База данных:**
- MySQL с нормализацией
- Индексы для производительности
- Foreign keys для целостности

### Монетизация

Freemium модель:
- Бесплатный доступ к базовым функциям
- Ограничение на количество привычек
- Реклама для free-пользователей
- Подписка 299/599 руб/мес

### Администрирование

Панель администратора:
- Управление пользователями
- Изменение тарифов вручную
- Бан/разбан
- Статистика по платформе

### Выводы

Проект демонстрирует:
- Навыки работы с MVC
- Реализацию системы подписок
- Безопасность (XSS, SQL injection)
- Адаптивный дизайн
- Бизнес-логику монетизации

### Перспективы

- Интеграция платежей
- Email-уведомления
- Мобильное приложение
- Социальные функции (друзья, лидерборды)
