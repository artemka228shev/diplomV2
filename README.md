# Habitify — Трекер привычек

Веб-приложение для отслеживания и формирования полезных привычек.

## Установка

### 1. Требования
- OpenServer / XAMPP / любой веб-сервер с PHP 8.0+ и MySQL
- phpMyAdmin (для импорта БД)

### 2. Установка
- Скопировать папку проекта в `domains/` (OpenServer) или `htdocs/` (XAMPP)
- Открыть phpMyAdmin, перейти на вкладку SQL
- Открыть файл `database/schema.sql`, скопировать содержимое и выполнить
- Если нужно сменить пользователя/пароль MySQL — отредактировать `app/config/database.php`

### 3. Запуск
- Перезапустить OpenServer / веб-сервер
- Открыть в браузере домен (например `http://diplom/`)
